## Reporting a Vulnerability

We take the security of this project seriously. If you find any security vulnerability, please do NOT open a public issue.

Instead, please report it privately:

1. Email the details to **arunkumarbind150720@gmail.com**
2. Include steps to reproduce the issue
3. We will respond within 48 hours to acknowledge your report

We will verify the issue and patch it as soon as possible.
